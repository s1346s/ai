1. HTML 기본 구조
   - <!DOCTYPE html> : HTML 문서 선언
   - <html>, <head>, <body> 기본 골격
   - <meta charset="utf-8"> : 인코딩 설정 (한글 깨짐 방지)
   - <title> : 문서 제목

2. CSS 적용
   - <style> 내부 스타일 or <link href="파일.css"> 외부 스타일
   - 기본 문법 : 선택자 { 속성:값; }
   - 예: body { background-color:#FF00FF; }

3. 링크와 이미지
   - <a href="URL">링크</a>
   - <img src="경로" alt="설명" width/height 로 크기조절
   - 외부URL / 내부이미지(ch02_img 폴더) 둘 다 가능
   - <hr /> : 수평선, color 속성으로 색 변경 가능

4. 텍스트 관련 태그
   - <h1>~<h6> : 제목태그(자동개행)
   - <p> : 단락태그 (자동 줄바꿈 + 여백)
   - <b>, <strong> : 진하게
   - <i> : 이탤릭체
   - <ins> : 밑줄
   - <del> : 취소선
   - <sub>, <sup> : 아래첨자, 위첨자
   - <span> : 인라인 스타일 지정용 태그
   - &lt;, &gt;, &nbsp; : 특수문자 표기

5. 목록 태그
   - <ol> : 순서 있는 목록 (type="1", "a", "A", "i", "I")
   - <ul> : 순서 없는 목록
   - <li> : 항목 (list item)

6. 표(table) 태그
   - <table>, <tr>, <th>, <td> 기본 구조
   - <caption> : 표 제목
   - colspan, rowspan : 셀 병합
   - CSS로 디자인 : 배경색, 여백, 정렬 등 지정
     예) table{background-color:aqua;} tr:hover{background-color:pink;}
   - border-collapse:collapse; 로 경계선 합치기

7. 기타
   - <pre> : 입력한 대로 출력(공백, 줄바꿈 유지)
   - id 속성 : 특정 위치로 이동할 때 (#top)
   - <a href="#top">맨위로</a> : 내부링크 기능

8. 이미지(img) 태그

   - `<img>` : 이미지를 삽입할 때 사용
   - `src` : 이미지 파일 경로 또는 URL 지정
   - `alt` : 이미지가 표시되지 않을 때 대체 텍스트 표시
   - `width`, `height` : 이미지 크기 조절
   - 예) `<img src="ch02_img/sulhyun.jpg" alt="설현 이미지" width="200" height="150" />`

9. 오디오(audio) 태그

   - `<audio>` : 오디오(음악, 음성 등)를 재생할 때 사용
   - `src` : 오디오 파일 경로 지정
   - `controls` : 재생/일시정지 등 컨트롤 표시
   - `loop` : 반복 재생
   - `preload` : 미리 로드 설정 (`auto`, `metadata`, `none`)
   - 예) `<audio src="ch02_sound/stevejobs1.mp3" controls preload="auto" loop></audio>`

10. 비디오(video) 태그

- `<video>` : 동영상을 삽입할 때 사용
- `src` : 비디오 파일 경로 지정
- `controls` : 재생, 일시정지 등의 버튼 표시
- `poster` : 영상 재생 전 표시할 대표 이미지 지정
- `width`, `height` : 영상 크기 조절
- 예) `<video src="ch02_sound/sonatina.mp4" controls poster="ch02_img/sulhyun.jpg" width="200"></video>`
