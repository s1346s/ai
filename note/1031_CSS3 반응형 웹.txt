1. test_toggle.html
- 체크박스로 메뉴 ON/OFF
- #toggle:checked ~ #lnb {display:block;}

2. ex.html / ex.css
- 모바일 뉴스 페이지
- header: 오렌지색 배경, 로고+검색창
- lnb: 6개 메뉴 (체크시 표시)
- section1~3: 뉴스 이미지, 제목, 말줄임
- footer: 로고 + 주소

3. link.html
- <link>로 외부 CSS 연결
- 화면용(screen), 인쇄용(print) 구분 가능

4. 규칙.html
- @import로 CSS 불러오기
- @media screen / print 사용
- 화면: 초록 배경 + 흰 글씨
- 인쇄: 초록 글씨 + 큰 폰트

5. CSS 파일
- title.css : 제목 중앙정렬, 밑줄
- content-screen.css : 초록배경, 흰글씨
- content-print.css : 흰배경, 초록글씨
- footer.css : 하단 중앙정렬

6. 반응형웹.html
- @media로 기기별 스타일 변경
- PC(800px), Tablet(연두색), Mobile(분홍색)
- viewport 설정 필수
